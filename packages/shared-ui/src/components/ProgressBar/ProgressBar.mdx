## Overview

ProgressBar is a component that displays the progress of a multi-step process. It is used to indicate the current step in a sequence of steps.

## Implementation
Usage example using jsx

```jsx
const ExampleUsage = () => {
  const [currentStep, setCurrentStep] = useState(0);

  const steps = [
    { label: 'Step 1', ariaLabel: 'Step 1 of 3' },
    { label: 'Step 2', ariaLabel: 'Step 2 of 3' },
    { label: 'Step 3', ariaLabel: 'Step 3 of 3' },
  ];

  return (
    <ProgressBar
      steps={steps}
      currentStep={currentStep}
      showLabels={true}
    />
  );
};
```
