## Implementation

Usage example using jsx:

```jsx
const ExampleCard = () => {
  return (
    <Card
      cardTitle="Example Card"
      description="This is an example card description that shows how to use the component."
      buttonTitle="Click Me"
      imageSrc="/example-image.jpg"
      imageAlt="Example"
      showImage={true}
      showDescription={true}
      showButton={true}
      onClick={() => console.log('Card clicked')}
    />
  );
};
```

## Usage Guidelines

➡️ Square variant must contain image vs horizontal variant which can optional hide/ show image

➡️ Text can be center aligned in horizontal variant

➡️ Min width is 288px on desktop or wide screens otherwise should be set to fill width of parent container
