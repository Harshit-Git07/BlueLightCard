## Composition
Having separate components allows a more complex implementation to be achieved than is practical via complex component props.

The example below uses RadioButton components for the first 4 radio items, but uses RadioButtonInput separately for penguin allowing a custom layout.

<Canvas of={RadioButton.CompositionExample} />



```jsx
<>
  <RadioButton name="dc-characters" id={'batman'} checked>Batman</RadioButton>
  <RadioButton name="dc-characters" id={'joker'}>Joker</RadioButton>
</>

```
